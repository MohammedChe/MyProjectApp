Just News

Just News is a typographic based responsive news site The aim of this project was to create a news website with no graphics or color.

The site is responsive which means that it will respond to the size of the browser. It does this by resizing some elements and bringing other elements closer together or by arranging them vertically. Check it out

The website re-aligns and adjusts the elements automatically, depending on the size of the screen.
